# Реализация класса "Дробь" на основе типа данных mpz_t. 

## Описание
В программе создан класс Fration и продемонстрирована работа с ним.

Были созданы методы:

- Set(num, den) - установка значений числителю и знаменателю
- reduce() - сокращение дроби
- toDecimal(digitNumber = 26) - формирование десятичной дроби с digitNumber знаками за запятой

А так же были перегружены операторы =, +, -, \*, /, по правилам математики, и оператор <<, который необходим для вывода данных на экран и в файл.

### Выходные данные
После работы программы создаётся файл, в котором записаны дроби и результаты арифмитических операций над ними.

### Запуск программы (linux mint)
Для начала нужно установить библиотеку gmp:
```
sudo apt install libgmp-dev
```
После чего можно компилировать и запускать программу.
```
g++ fraction.cpp -o fract -lgmp
./fract2
```
